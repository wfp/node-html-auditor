# node-pa11y-auditor

## Installation

```
npm install
npm link
```

## Usage

### Fetch HTML

```
html-fetch --uri [URI]
```

### Accessibility audit (a11y, WCAG)

```
a11y-audit --path [path/to/folder]
a11y-audit --path [path/to/folder/file.html]
a11y-audit --path [path/to/folder/file.html path/to/folder/file-1.html ... ]
